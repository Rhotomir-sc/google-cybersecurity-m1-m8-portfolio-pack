# Google Cybersecurity Portfolio Pack

This repository contains structured notes and evidence from the Google Cybersecurity Certificate courses.

I use this repository to document how I learned cybersecurity fundamentals, including security frameworks, risk management, network security, Linux, SQL, detection and response, Python basics, and incident documentation.

---

## Purpose of This Repository

This repository answers one main question:

> How did I learn cybersecurity fundamentals in a structured way?

The goal is to keep my Google Cybersecurity learning organized as a portfolio pack, not just as course completion notes.

This repository is designed to show:

- Cybersecurity fundamentals
- Security frameworks and risk concepts
- Basic GRC awareness
- Linux and SQL practice
- Network security foundations
- Detection and response basics
- Incident documentation practice
- Python-based security task basics

---

## Course Areas

The Google Cybersecurity Certificate covers eight main course areas:

1. Foundations of Cybersecurity
2. Manage Security Risks
3. Networks and Network Security
4. Linux and SQL Tools
5. Assets, Threats, and Vulnerabilities
6. Detection and Response
7. Automate Cybersecurity Tasks with Python
8. Prepare for Cybersecurity Jobs
