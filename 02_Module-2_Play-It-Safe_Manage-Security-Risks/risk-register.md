# Risk Register (Module 2)

| Risk | Asset | Threat | Likelihood | Impact | Mitigation (Control) |
|------|-------|--------|------------|--------|----------------------|
| Weak passwords | User accounts | Brute force / credential stuffing | Medium | High | Strong password policy + lockout + MFA where possible |
| Privilege misuse | Local admin group | Unauthorized admin access | Low-Med | High | Least privilege + audit changes + admin group monitoring |
| Unpatched system | OS/Apps | Exploitation of known vulns | Medium | High | Update schedule + basic inventory + vulnerability awareness |
| Phishing | User | Credential theft | Medium | High | Awareness + email hygiene + verify links/attachments |
| Data exposure | Personal files | Accidental sharing | Low | Medium | Access control + backups + safe storage practices |
